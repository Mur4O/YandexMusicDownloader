# Обёртка библиотеки [Yandex Music API](https://yandex-music.readthedocs.io/en/main/index.html)

Имеет два режима работы (выбираются в коде файла main.py):
- скачать только любимые треки в одну папку
- скачать альбомы содержащие любимые треки

Результаты скачивания попадают в папку src/Музыка

Также программа умеет отслеживать уже скачанные песни (лежащие в изначальной папке) и докачивать недостающие
